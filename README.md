# 🛡️ Program Architect V1.0 
### 基于现代力量训练科学的非线性周期引擎

**"连接实验室理论与健身房杠铃的逻辑桥梁。"**

---

## 📖 项目简介
**Program Architect V1.0** 是一款专为中高级力量举（Powerlifting）训练者设计的数字化教练工具。本系统拒绝死板的线性增长计划，而是深度集成了全球力量训练领域的三大核心著作逻辑：

1.  **《力量训练原理》(Scientific Principles of Strength Training)** —— 核心架构与疲劳管理。
2.  **《肌肉与力量训练金字塔》(The Muscle & Strength Pyramid)** —— 容量与强度配比。
3.  **《主宰方法 2.0》(The Juggernaut Method 2.0)** —— 非线性波浪周期设计。

## 🚀 核心功能
*   **非线性波浪式负荷 (Non-Linear Wave Loading)**：通过周次间的强度波动，在“积累-转化-实现”三个阶段中循环递进，有效避免训练撞墙期。
*   **智能化目标逆推算法**：用户设定周期终点目标后，系统利用线性插值自动计算每周的理论负载。
*   **真实世界配重修正**：内置 `round_to_plates` 逻辑，确保所有生成的重量均符合 2.5kg 杠铃片最小增幅。
*   **动态周次导航**：通过交互式滑块，实时解锁每一周的详细训练方案，包括主项变式建议与辅助项容量调整。
*  
## 🛠️ 使用指南
1.  **档案输入 (Profile)**：在侧边栏输入你当前的深蹲(S)、卧推(B)、硬拉(D) 1RM 数据。
2.  **目标设定 (Goal)**：设定你的周期目标及期望的训练周数（建议 12-20 周）。
3.  **可行性评估**：系统将实时反馈目标的激进程度，帮助你优化预期。
4.  **计划执行**：拖动时间轴滑块，获取当前周的组数、次数、重量及辅助项安排。
5.  **深度研究**：查阅页尾的参考文献与动作词典，深化对训练原理的理解。
